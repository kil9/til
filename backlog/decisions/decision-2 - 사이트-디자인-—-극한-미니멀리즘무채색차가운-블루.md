## Context

사이트 전체 디자인 방향을 검토(구 PLAN T-19 → T-23, 2026-07-12 확정). 1차 3안 및 이후 세부안 제안은 사용자가 전부 반려하고 미니멀 방향을 직접 지정.

## Decision

**극한 미니멀리즘** 을 사이트 공통 디자인 언어로 확정.

- 무채색(순백/근흑 쿨 뉴트럴) + 차가운 블루 액센트(`#1A5FC8` / 다크 `#82B1F0`).
- 카드 껍데기(배경·테두리·그림자·라운드) 전부 제거.
- 날짜(모노) + 주제를 왼쪽 거터로 두는 **단일 리스트**(본문 폭 720px, 기본 16px).
- 필터 칩은 텍스트 버튼(활성=블루 밑줄), `.path` 표시 제거, eyebrow 제거.
- 마크업·클래스·`data-*`·인라인 JS(정렬/아카이브/필터)와 `.ai-summary` 구조는 그대로 유지 → **nuc14 `inject.py` 는 무수정**.

AGENTS.md 런북 §2 의 신규 페이지 공통 셸 템플릿(동일 토큰)이 이 디자인의 정본 구현이다.

## Consequences

- 신규 페이지는 AGENTS.md 공통 셸을 기본으로 쓴다(artifact 자체 스타일 페이지는 원본 존중하되 favicon·OG·beacon 필수).
- 정보 밀도는 카드→리스트 전환으로 확보. 항목 수가 크게 늘어 단일 리스트가 길어지면 레이아웃 재평가(구 T-18 다단 칼럼 안은 이 결정으로 폐기됨).
