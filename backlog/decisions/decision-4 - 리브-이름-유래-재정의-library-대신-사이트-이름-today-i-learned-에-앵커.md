## Context

TASK-1 에서 확정한 doc-3 는 이름 '리브'의 유래를 "library 의 lib 에서 따온 이름"으로 기록했다. 직함이 '서고지기'였고 "서고"가 결과물이 쌓이는 곳을 부르는 시그니처였으니 일관된 설정이었다.

TASK-6 에서 사용자 피드백으로 '서고지기' 직함과 서고·도서관 비유를 사이트 전반에서 걷어내고 실제 역할대로 'AI 지식 큐레이터'로 정리하기로 했다. 그러면서 풀네임을 '리브 투데이'(Liv Today)로 확정했다. 이때 유래만 library 로 남겨두면, 본문에서 비유를 지워도 정본이 캐릭터를 계속 도서관에 묶어두게 된다. 이름 자체(리브)는 이미 설정화 시트·발행 페이지·루트 인덱스에 박혀 있어 바꾸지 않는다.

## Decision

이름은 '리브'로 유지하되 유래를 재정의한다.

- 리브(Liv): 짧고 부르기 쉬운 외래어풍 이름. library 근거는 폐기한다.
- 투데이(Today): 사이트 이름 `today i learned` 에서 딴 성.
- 합쳐서 Liv Today — 사이트 이름의 "오늘"을 그대로 이름에 얹은, 오늘 알게 된 것을 그날 정리해 올리는 화자라는 뜻.

## Consequences

- doc-3 §1 의 유래 항목을 위 내용으로 교체한다. 완료 이력(doc-2, 마일스톤 m-0, 완료 태스크 파일, 아카이브 PLAN.md)은 그때의 판단 기록이므로 소급하지 않는다.
- 이름을 유지하므로 설정화 시트 8컷·비주얼 스펙은 그대로 유효하다. 재생성 없음.
- 이미 발행된 liv-archivist 페이지에는 옛 유래가 남지만 TASK-7 에서 통째로 대체된다.
