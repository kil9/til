## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
루트 인덱스 사이드바에는 사람 관리자 블록(.human, kil9)이 있지만(TASK-8 아래 커밋 24e3536), 리브 소개 페이지(liv-today/index.html)에는 대응물이 없다. 같은 아바타 블록을 그 페이지 하단에도 붙여 봤으나 리브의 1인칭 자기소개 흐름에 맥락 없이 떠 있어 폐기했다. 블록이 아니라 리브가 직접 말하는 문장으로 녹이는 편이 그 페이지 성격에 맞는다는 결론.

주의 — 이 페이지는 이미 '관리자님'을 6곳에서 언급한다. 다만 전부 과거의 설정 결정(화자를 세우기로 고르심, 문체 규칙을 정하심, 조사를 던져 주심)에 대한 것이고, '이 아카이브를 운영하는 주체가 관리자님'이라는 관리자 프레이밍은 없다. 그 구멍만 메우는 것이 이 태스크다. 단순히 관리자님을 한 번 더 언급하면 기존 6곳과 중복돼 no-op 이 된다.

인터뷰 확정 사항: 호칭은 '관리자님' 유지 — 실명 kil9 는 드러내지 않는다(현재 본문에 kil9 는 og:url 에만 등장). 링크도 걸지 않는다(하단·인덱스 footer 에 이미 GitHub 링크가 있다). 넣을 자리는 구현자 재량 — 섹션은 이름부터/제가 왜 여기 있는가/하는 일/어디까지 나설지/그럼 5개.
